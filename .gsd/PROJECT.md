# FitLife — 종합 헬스 & 피트니스 앱

## What
운동(근력+러닝), 식단/영양, 수면, 에너지 관리를 하나의 앱으로 통합한 개인 건강 관리 플랫폼.
iOS 네이티브(SwiftUI) + Web(Next.js) 듀얼 플랫폼.

## Who
개인 프로젝트. 사용자: 본인(CrossFit + 10K 마라톤 40분대 목표, 운동일 2,300kcal / 휴식일 1,900kcal, 단백질 150g/일).

## Benchmarks
- **근력 운동**: Fitbod (AI 개인화, 근육 회복 맵, progressive overload), Strong (세트/렙 로깅, PR 트래킹)
- **러닝**: Nike Run Club (GPS, 트레이닝 플랜, 오디오 코칭, 게이미피케이션)
- **식단/영양**: MyFitnessPal (1,800만+ 식품 DB, AI 매크로 트래커, 바코드 스캔)
- **수면/에너지**: Samsung Health (수면 단계 분석, Sleep Coaching, Energy Score)
- **홈트레이닝**: Nike Training Club (185+ 무료 워크아웃)

## Tech Stack
- **iOS**: SwiftUI + SwiftData + HealthKit + CoreLocation + MapKit
- **Web**: Next.js + TypeScript + Tailwind CSS
- **Backend**: Firebase (Auth + Firestore + Cloud Functions) — 공유 백엔드
- **AI**: Vision API + Claude API (식단 분석)

## 5 Modules
1. 워크아웃 트래커 (근력 + CrossFit)
2. 러닝 트래커 (GPS, 10K 플랜)
3. 식단 & 영양 관리 (칼로리, 매크로, 한국 식품 DB)
4. 수면 & 에너지 관리 (수면 트래킹, 에너지 스코어)
5. 대시보드 & 게이미피케이션 (통합 리포트, 배지, 위젯)
