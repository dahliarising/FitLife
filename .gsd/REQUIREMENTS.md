# Requirements

## Active

- **R001**: iOS 앱은 SwiftUI로 구축하며 iOS 17+ 지원 — `validation: Xcode 빌드 + 시뮬레이터 실행`
- **R002**: Web 앱은 Next.js + TypeScript로 구축 — `validation: npm run build 성공 + 브라우저 확인`
- **R003**: Firebase 백엔드로 iOS/Web 데이터 동기화 — `validation: 양 플랫폼에서 동일 데이터 확인`
- **R004**: 근력 운동 로깅(세트/렙/무게) 및 progressive overload 추천 — `validation: 운동 기록 저장 및 다음 세션 추천 동작`
- **R005**: GPS 기반 러닝 트래킹(페이스, 거리, 고도, 스플릿) — `validation: 실제 GPS 트래킹 동작`
- **R006**: 칼로리/매크로 트래킹 (운동일 2,300kcal / 휴식일 1,900kcal 자동 전환) — `validation: 스케줄 기반 자동 목표 변경`
- **R007**: 한국 음식 중심 식품 데이터베이스 — `validation: 주요 한식 50개+ 검색 가능`
- **R008**: 수면 트래킹 및 에너지 스코어 (HealthKit 연동) — `validation: HealthKit 수면 데이터 읽기 + 점수 계산`
- **R009**: 통합 대시보드 (운동/식단/수면/에너지 한눈에 보기) — `validation: 4개 영역 데이터가 하나의 화면에 표시`
- **R010**: 게이미피케이션 (스트릭, 배지, 챌린지) — `validation: 연속 운동일 카운트 + 배지 획득 동작`
- **R011**: 10K 마라톤 40분대 트레이닝 플랜 — `validation: 주별 러닝 프로그램 자동 생성`
- **R012**: 주간 운동 스케줄 (월:어깨/팔, 화:하체, 목:가슴/등, 금:코어) — `validation: 요일별 프로그램 자동 표시`
